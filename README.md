## 關於油價 Bot
使用Node.js架設於Heroku進行開發，搭配Uptime Robot觸發Heroku<br />

透過使用者加入好友事件(follow)<br />

存取Line的Token至資料庫mLab(MongoDB)<br />

使用cheerio css select 抓取資訊<br />

## 特色
每周五及日晚上六點*自動推送下周油價*給使用者<br />

使用者點選圖文選單中的即時或下周油價時<br />

一樣可以達到相同效果<br />
